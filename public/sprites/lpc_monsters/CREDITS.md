# lpc_monsters — attribution

26 monster sprite sets, each pre-baked (offline PIL composite: body sheet +
optional tint + monster head sheet; no repainting) from the Universal LPC
Spritesheet Character Generator assets:
https://github.com/liberatedpixelcup/Universal-LPC-Spritesheet-Character-Generator

Each `<id>/` folder holds standard LPC-geometry sheets:
`walk.png` (9f x 4 dir), `hurt.png` (6f death collapse, 1 row), and one
attack sheet — `slash.png` (6f x 4), `thrust.png` (8f x 4) or
`spellcast.png` (7f x 4).

Parts used:
- Bodies: male / muscular / skeleton / zombie — authors incl. Stephen
  Challener (Redshrike), Johannes Sjölund (wulax), bluecarrot16,
  Benjamin K. Smith (BenCreating), Evert, Eliza Wyatt (ElizaWy),
  MuffinElZangano, Durrani, Pierre Vigier (pvigier), Matthew Krohn (makrohn),
  Lanea Zimmerman (Sharm), and others per the ULPC CREDITS.csv.
- Heads: orc, goblin, troll, minotaur, lizard, wolf, boarman, rat, mouse,
  vampire, frankenstein, alien, jack (pumpkin), wartotaur, pig, skeleton,
  zombie — authors incl. bluecarrot16, Benjamin K. Smith (BenCreating),
  Stephen Challener (Redshrike), Thane Brimhall (pennomi), laetissima,
  and others per the ULPC CREDITS.csv.
